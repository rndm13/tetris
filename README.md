# _Tetris_ !
## Made with [FTXUI]
### Controls:
Move tetrominoes with asd keys.
Rotate with q and e.
Drop with w and save with c.
### Installation:
CMake version 3.10 is required.
```
git clone https://github.com/rndm13/tetris.git
mkdir tetris/build
cd tetris/build
cmake ..
cmake --build .
```
